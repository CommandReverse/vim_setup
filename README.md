# VIM Setup

My "simple" vim setup

## How to

Create folder ~/.config/nvim/  
Download NodeJs ( version > 12 )  
Download Vimplug into nvim/ folder  
Download OceanicNext and copy the colors/ folder into nvim/colors/  
Inside of init.vim file run :PlugInstall  
:CocInstall modules  

Where modules  
coc-html  
coc-css  
coc-markdownlint  
coc-json  
coc-emmet  
coc-pyright
